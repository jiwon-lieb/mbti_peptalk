🌟 MBTI Pep Talk API 🌟

Your personalized pep talk API for all MBTI types!

📚 Introduction

The MBTI Pep Talk API provides personalized pep talks for each MBTI type. You can customize the mood to get pep talks for moments of sadness, anger, or happiness.

📡 API Endpoints

Endpoint Method Description
/api/peptalk/:mbti GET Returns a random pep talk for the specified MBTI (default mood). Example: /api/peptalk/INTJ
/api/peptalk/:mbti?mood=sad GET Returns a pep talk for the specified MBTI when they feel sad. Example: /api/peptalk/ENTP?mood=sad
/api/peptalk/:mbti?mood=angry GET Returns a pep talk for the specified MBTI when they feel angry. Example: /api/peptalk/INFJ?mood=angry
/api/peptalk/:mbti?mood=default GET Returns a pep talk for the specified MBTI in their default mood. Example: /api/peptalk/ISFP?mood=default
